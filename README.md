# bio-heuristic-datasets

## Descrição

bio-heuristic-datasets é uma ideia que se concretiza neste projeto de desenvolver e aperfeiçoar meus conhecimentos de algoritmos baseados na natureza aplicados em data science na geração de datasets.

## Versão 0.0.0.1

A versão 0.0.0.1 é a versão mais simples e a que dá o pontapé inicial com o uso de algoritmos genéticos para a geração de dados de compras comuns com alguns casos de transações de fraudes.

A ideia é seguir os pequenos passos:

[] Decidir 5 a 10 colunas iniciais para representar uma transação
[] Revisar e aplicar algoritmos genéticos para o caso de transações de um banco (fraudes + transações normais)
[] Definir o percentual correto para otimizar ou evoluir para uma distribuição mais comum encontrada em bancos
[] Visualizar os dados em 2 ou 3 gráficos simples

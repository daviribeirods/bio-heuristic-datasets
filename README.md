# bio-heuristic-datasets

# Versão alfa 0.0.0.1

Esta versão alfa 0.0.0.1 traz as ideias gerais, o brainstorm, o motivador e objetivo deste projeto. A ideia original veio agora mas pode ser modificada conforme a necessidade e o desejo de mudar o rumo do projeto no futuro como melhor for o contexto e desejo do autor.

Meu nome é Davi, e quero fazer este projeto para reaprender a desenvolver algoritmos genéticos, colonia de formigas, algoritmos de otimização (encontrar máximos e mínimos locais), usando também heurísticas e meta-heurísticas, adaptando para a geração de dados fictícios do ramo de finanças e ramos relacionados.

O motivo é simples: reaprender esses algoritmos, e de quebra, gerar datasets meus para projetos futuros. A versão 0.0.0.1 trará códigos que funcionam, com poucas ou apenas 2 variáveis, com o intuito de ter uma versão mínima, embrionária e simplória mas que esteja pronta. Para esta versão, eu escolho:

[] Desenvolver o algoritmo genético - criar um cromossomo de alguma forma, quem sabe um dataset no ramo de finanças com 5 variáveis bem simples
[] Aplicar algoritmos genéticos
[] Criar a função necessária com o objetivo que eu quero para o dataset
[] Ao final, exportar o arquivo em .csv para que alguém - no caso eu mesmo, possa analisar e aplicar estatística, machine learning e etc.

No futuro, farei versões melhores, que gerem dados de outras naturezas (exemplo: se eu gerei um dataset de fraudes de cartão de crédito, no futuro farei outro dataset de outra natureza e que respeite outra curva para aquele tipo de dado).
